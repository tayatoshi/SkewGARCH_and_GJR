# SkewGARCH_and_GJR
以下の実装とML推定。
- 標準正規分布の場合のGARCH(1,1)
- Hansen(1994)のskew-t分布の場合のGARCH(1,1)
- 標準正規分布の場合のGJR

## 環境
- MacBook Pro(M2), macOS Monterey 12.4
- Python 3.10.6
    - matplotlib 3.5.2
    - numpy 1.23.1
    - pandas 1.4.3
    - scipy 1.9.0
